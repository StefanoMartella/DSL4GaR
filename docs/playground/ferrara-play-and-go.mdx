# Ferrara Play&Go

**Ferrara Play&Go** is a an example of gameful system which aims to make the use of sustainable means of transportation enjoyable and rewarding.
All the rules of the game have been defined using the DSL4GaR.

The following video shows the workflow adopted to generate, simulate and deploy all the rules of the game:
